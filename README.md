# Text Analysis Tool

## 📖 Overview
A text analysis tool built for the Buildables AI Fellowship Week 1 assignment.  
It demonstrates:
- Summarization with Gemini LLM
- Tokenization with GPT-2 & BERT
- Token usage & cost estimation
- Language detection
- Sentiment analysis
